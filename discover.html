Question-1: What are the different ways to select an element in the DOM?

We can select element in dom using many method :
1. By document.getElementById('id');
2. By document.getElementsByClassName('class');
3. By document.querySelector('class');
4. By document.querySelectorAll('class');
5. By document.getElementsByTagName('tag');
6. By document.getElementsByName('name');

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference:
1. innerHTML can can change and set inside html tags.
2. innerText can set inside text.
3. textContent can set all type of text.
